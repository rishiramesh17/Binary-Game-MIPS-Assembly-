How to Run the Program
1. Download and launch MARS / Mips assembler
2. Load all 4 files in the same folder into a new Mips program
3. Go to file and select main.asm
4. Click the wrench icon / Assemble
5. Click the green play button or the Go button
6. The program will start in the console window at the bottom

How to Play the Game
1. Start the Game - The game displays a welcome message and starts at Level 1 with your score at 0.
2. Answer the Problem - You will be given a decimal number or an 8 digit binary number, and you must convert it to the other
3. Input Rules - Decimal must be 0-255, and binary must be exactly 8 digits of 0s and 1s only.
4. Scoring - Each correct answer earns 10 points, and wrong answers show the correct solution.
5. Completing the Game - After finishing all 10 levels, view your final score and choose to play again (1) or exit (0).
